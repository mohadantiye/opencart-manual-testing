# Test Plan – OpenCart Demo Website

## Objective

To verify the functionality, usability, and input validation of the OpenCart demo website.

## Scope

The following features will be tested:

* User Registration and Login
* Product Search
* Add to Cart functionality
* Checkout process
* Contact and Newsletter forms
* Session management

## Testing Types

* Functional Testing
* Negative Testing
* Exploratory Testing

## Test Environment

* Browser: Google Chrome
* Device: Laptop
* OS: macOS

## Entry Criteria

* Website is accessible
* Test data is available

## Exit Criteria

* Core features have been tested
* Bugs have been identified and documented

## Deliverables

* Test Plan
* Test Cases
* Bug Reports
* Screenshots
* Test Summary Report
